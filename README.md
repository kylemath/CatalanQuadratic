## Description
An interactive web application visualizing the connection between hexagon triangulations and polynomial solutions, inspired by the paper "A Hyper-Catalan Series Solution to Polynomial Equations, and the Geode" by Norman Wildberger and Dean Rubine. This application provides an intuitive geometric interpretation of how polynomial solutions relate to triangulations of regular polygons.

## Features
* Interactive 3D visualization of hexagon triangulations
* Visual representation of polynomial roots and their geometric connection
* Support for exploring different triangulation patterns
* Modern UI with crystal-like aesthetics

## Current Implementation
The application currently focuses on the quadratic equation case (1-x-xÂ² = 0), demonstrating how its solutions relate to hexagon triangulations. The visualization shows:

1. A regular hexagon with different triangulation patterns
2. A number line displaying the roots of the polynomial equation
3. Visual connections between the triangulation patterns and the polynomial roots
